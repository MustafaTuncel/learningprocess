CSS Descriptions

CSS KULLANIM ŞEKİLLERİ
    CSS üç şekilde kullanılabilir :

    1 - Internal CSS - Sayfa İçi Tanımlama

    Yazılan CSS kodları sadece tek bir sayfayı ilgilendiriyorsa kullanılır.

      Head bloğu içerisinde yazılan < style> tanımlamalarıdır.
      < head>
      < style> body{color: pink;} < style>
      < /head>       gibi.

    2 - InLine CSS - Satır İçi Tanımlama

    Yazılan CSS kodları sadece tek bir satırı ilgilendiriyorsa kullanılır.
    InLine yazılan CSS kodları Internal CSS kodlarını ezer.

      Satır içerisinde yazılan < style> tanımlamalarıdır.
      < p id="ilkparagraf" style="border: 1 solid #555" ;> Lorem, ipsum dolor sit amet consectetur adipisicing elit. Quas, quasi? < /p> gibi.

    3 - External CSS - Sayfa Dışı Tanımlama

    Birden çok html dosyasına CSS ile etki etmek istendiğinde kullanılır.
    Internal yada InLine yazılan CSS kodları External CSS kodlarını ezer.

      Projenin bulunduğu klasör içerisinde style.css adında bir dosya oluşturulur.
      CSS tanımlamaları bu dosyanın içerisine yazılır ve dosya içerisinde tekrardan < style> tagi kullanılmaz.

      Daha sonra çalışılan html dosyası içerisinde style.css dosyasıyla şu şekilde bağlantı kurulur ;
      < link rel="stylesheet" href="style.css">

    CSS Selectors
      Group Selectors:
      • * = tüm etiketler
      • p = tüm paragraf etiketleri
      • div.classadı = belli sınıftaki divleri seçer(divin kendisi)
      • div .classadı = belli sınıfları seçer(divin altındaki sınıflar)
      • div p = div içindeki tüm p etiketleri
      • div>p = sadece üst etiketi div olan p etiketleri
      • div,p = tüm div etiketleri ve tüm p etiketleri
      • p~div = p ile aynı seviyedeki div etiketleri ve p etiketlerinden sonra gelen div etiketleri
      • p+div = p etiketinden sonra gelen ilk div etiketleri

      Attribute Selectors:
      • p[title] {border: 1px solid gray; padding: 5px;} p elementlerindeki title attributelerinin tümünü seçer
      • p[title="third paragraph"] {background-color: gold;} p elementlerindeki title attributelerinde geçen (third paragraph) title adını seçer
      • p[title~"third"] {background-color: red;} p elementlerindeki title attributelerinin içinde geçen (third) değerin bulunduğu titleları seçer (= yerine ~ kullanılır)
      • a[href^="http"] {color: red;} a elementlerindeki href attributelerinin başında geçen (http) değerin bulunduğu linkleri seçer
      • a[href$="com"] {color: aqua;} a elementlerindeki href attributelerinin sonunda geçen (com) değerin bulunduğu linkleri seçer
      • a[href="https://www.example.net"][target="_blank"] {color: yellow;} a elementlerindeki href ve target attributeleri içinde geçen (https://www.example.net ve _blank) değerlerin bulunduğu linkleri seçer
      • input[type="text"] {display: block; width: 150px; background-color: red;} input elementlerindeki type attributelerinin içinde geçen (text) değerin bulunduğu inputları seçer
      • input[type="button"] {background-color: green; color: white; padding: 8px 16px;} input elementlerindeki type attributelerinin içinde geçen (button) değerin bulunduğu inputları seçer
      • img[src="https://www.example.com"] {color: orange;} img elementlerindeki src attributelerinde geçen (https://www.example.com) değerin bulunduğu resimleri seçer
      • div[id="example1993"] div elementlerindeki id attributelerinde geçen (example1993) değerin bulunduğu div bloklarını seçer

      Pseudo Selectors
      Sayfa içerisindeki elementlerin bir parçasını seçmek istediğimizde Pseudo Elementlerini kullanırız.
      Sayfa içerisindeki elementlerin altında bulunan attributeleri seçmek istediğimizde Pseudo Classlarını kullanırız.

        Pseudo Elements
        • p::first-line {font-weight: bold; color: brown; word-spacing: 2px; letter-spacing: 1px;}
        • p::first-letter {font-size: 48px;}
        • p::before {content: "-deneme"; color: blueviolet; background-color: yellow; width: 60px;}
        • p::after {content: "-deneme"; color: blueviolet; background-color: yellow; width: 60px;}

        Pseudo Classes
        • ul li {padding: 10px; margin: bottom 10px; list-style-type: none; width: 450px; border-radius: 50px;}
        • ul li:nth-of-type(odd) {border: 2px solid blue; margin: 5px; background-color: green; color: white;}
        • ul li:nth-of-type(even) {border: 2px solid red; margin: 5px; background-color: purple; color: white;}
        • ul li:first-child {background-color: yellow; color: blue;}
        • ul li:last-child {background-color: yellow; color: red;}
        • a:link {color: goldenrod;}
        • a:hover {color: green;}
        • a:active {color: red;}
        • a:visited {color: purple;}
        • div:hover span {display: block;}

      CSS Inherit

      a elementini baz alırsak ; kendisi üzerinde bulunan elementlerden (body>ul>li>a vb.) css kodu almaz.
      Bu türdeki elementlerin üst elementlerden attribute değerlerini alabilmeleri için inherit kodu kullanılır.

      < div id=example yada class=example >< /div>
      #example span {color: inherit;} yada
      .example span {color: inherit;} gibi


SAYFA İÇİ DÜZEN VE ETİKETLER

  • < style> etiketiyle oluşturulur.
  • Oluşturulmuş id'yi seçmek için #id adı yazılır.
  • Oluşturulmuş class'ı seçmek için .class adı yazılır.
  • Değerden sonra girilen !important tagi seçili attribute'e verilen değeri diğer genel tanımlamalardan üstün kılar.
    #ilkparagraf yada .ilkbaşlıkparagrafları {font-size: 40px!important;} gibi.

CSS STİLLERİ
    Background

    gradient efekti = to 'yön' gradient efektinin başlangıç noktası (efekt ters yöne doğru devam eder)
    Linear = giderek artan gradient efekti
    Radial = merkezleri tarafından tanımlanır
    Conic = bir merkez noktası etrafında döndürülmüş

    Multiple = red, orange, yellow, green, blue, purple... (çok renkli gradient)
    Transparency = rgba(0,0,0,0), rgba(0,0,0,1)) (şeffaflık / opaklık verme)
    Direction = to right [soldan sağa (tek yön)}
    Degree = 0deg, 90deg, 180deg, -90deg [alttan üste, soldan sağa, üstten alta, sağdan solda (tek yön)}
    Diagonal = to right bottom [sol üstten sağ alta (çift yön)}

    clip-path = arka plan üzerinde açıları kullanarak arka planı kırpar
    viewport = tarayıcı üzerinde gösterilen arka plan alanı
    vh = viewport yüksekliği
    vw = viewport genişliği

    background-size = viewport'un yani arka planın tarayıcıda gözüken kısmının ölçüsü
    background-position = arka planın viewport üzerindeki konumu

    • background-image: url ("")

      • background-image: ...-gradient(red, yellow) efekt verme (top to bottom default ayardır)
        • background-image: linear-gradient(to left top, rgba(0,0,0,0), rgba(0,0,0,0)), url(""); (gradient efekti verme)

    • background-size: cover (arka planın kalitesini bozmadan çerçeveye sığdırır)
    • background-position: center (arka planı konumlandırır)
    • background-repeat: repeat, no-repeat, repeat X (yatay), repeat Y (dikey)
    • background-color: renk adı / kodu, rgb kodu, rgba kodu
    • background-attachment: 'attachment value' (arka plan resmi akış ayarı)

      • background-attachment: scroll (arkaplan resmi sayfayla birlikte kayar) (varsayılan)
      • background-attachment: fixed (arkaplan resmi sabit kalır, sayfayla birlikte kaymaz)
      • background-attachment: local (arkaplan resmi, (elementlerin) öğenin içeriğiyle birlikte kayar)
      • background-attachment: initial (varsayılan değer kodu)
      • background-attachment: inherit (bu özelliği üst (elementten) öğesinden alır)
      • background-attachment: unset ?

    • clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%) (arka plan üzerinde açısal crop işlemleri)
    • height / width: 100vh / 100vw (viewport'un ne kadarlık alanında arkaplanın gösterileceği)

      • flex-direction: column (alt alta gelir ve satırı kaplar, sıralama üstten başlar)
      • flex-direction: column-reverse (alt alta gelir ve satırı kaplar, sıralama alttan başlar)

    • flex-wrap (container içerisindeki flex-itemlara responsive kazandırmak)

      • flex-wrap: nowrap (satıra sığmayan elemanların aynı satırda kalması) (varsayılan)
      • flex-wrap: wrap (satıra sığmayan elemanların alt satıra inmesi)
      • flex-wrap: wrap-reverse (satıra sığmayan elemanların üst satıra çıkması)

    • justify-content: (container içerisindeki flex-itemları yatay hizalamak)

      • justify-content: flex-start (sola hizalamak)
      • justify-content: center (ortaya hizalamak)
      • justify-content: flex-end (sağa hizalamak)
      • justify-content: space-between (sadece flex-itemların aralarında x1 boşluk bırakarak eşit şekilde hizalamak)
      • justify-content: space-around (başlangıç flex-itemından önce ve son flex-itemından sonra x1, flex-itemların aralarında x2 boşluk bırakarak hizalamak)
      • justify-content: space-evenly (flex-itemların her yerinden eşit şekilde boşluk bırakarak hizalamak)

        • WARNING !
        flex-direction'a varsayılan row değeri kaldırılıp column değeri verilmişse ;
        justify-content dikey olarak, align-items yatay olarak düşünülür.

    • align-items (container içerisindeki flex-itemları dikey hizalamak)

      • align-items: flex-start (flex-itemları yukarı hizalar ve değer verilen itemlar sabit boyutunda kalır, değer verilmeyen itemlar küçülür.)
      • align-items: center (flex-itemları ortaya hizalar ve değer verilen itemlar sabit boyutunda kalır, değer verilmeyen itemlar küçülür.)
      • align-items: flex-end (flex-itemları aşağı hizalar ve değer verilen itemlar sabit boyutunda kalır, değer verilmeyen itemlar küçülür.)
      • align-items: baseline (flex-itemlar yazı ve içeriklerin alt sınırından hizalanır)
      • align-items: stretch (flex-itema verilen height gibi bir attribute değerini diğer değer verilmemiş flex-itemlara uygular,
        2 yada daha fazla flex-itema aynı attribute verildiyse fazla olan baz alınır.)         (varsayılan)

        • WARNING !
        flex-direction'a varsayılan row değeri kaldırılıp column değeri verilmişse ;
        align-items yatay olarak, justify-content dikey olarak düşünülür.

    • align-self (tek bir flex-itemı hareket ettirmek)

      • align-self: flex-start (dikeyde üstten hizalar)
      • align-self: center (dikeyde ortaya hizalar)
      • align-self: flex-end (dikeyde alttan hizalar)
      • align-self: baseline (yazı ve içeriklerin alt sınırından hizalanır)
      • align-self: stretch (değer verilmeyenleri verilen maksimum değere eşitler)

    • order (flex-itemları başa almak, yerlerini değiştirmek)

      • order: -1
      • order: -2

        • WARNING !
        Align-self ile birlikte kullanıldı,
        verilen en büyük '-' değer en başa ve verilen en büyük '+' değer en sona gelir.

    • flex: 0 1 auto / % / px (tek satırda büyüme, daralma ve genişlik oranını ayarlamak)

      flex: 1 = flex-grow: 1
      flex: 0 1 = flex-shrink: 1
      flex: 0 0 auto / % / px = flex-basis: auto / % / px

      • flex-grow: 1 (flex-container büyütüldüğünde flex-itemların büyüme oranını ayarlamak)
      • flex-shrink: 1 (flex-container daraltıldığında flex-itemların daralma oranını ayarlamak)
      • flex-basis: auto / % / px (flex-container genişletildiğinde flex-itemların genişlik oranını ayarlamak)

        • WARNING !
        "X" bir itema 1, "Y" bir itema 2 değeri verildiğinde "Y" itemı "X" itemının 2 katı değer kazanır.

    • align-content (flex-item div grupları arasına mesafe vermek)

      • align-content: flex-start (satır gruplarını üstten hizalar)
      • align-content: center (satır gruplarını ortadan hizalar)
      • align-content: flex-end (satır gruplarını alttan hizalar)
      • align-content: stretch (satırı olduğu gibi kaplar) (varsayılan)
      • align-content: space-between (satır gruplarının aralarına eşit şekilde mesafe koyar)
      • align-content: space-around (başlangıç satır grubundan önce ve son satır grubundan sonra x1, satır gruplarının aralarına x2 mesafe bırakarak hizalar)
      • align-content: space-evenly (satır gruplarından önce, sonra ve satır gruplarının aralarına eşit şekilde mesafe bırakarak hizalar)
