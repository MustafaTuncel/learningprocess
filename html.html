1 - SAYFA YAPISI

< html>
< head>
< /head>
< body>
< /body>
< /html>

2 - SAYFA İÇİ DÜZEN VE ETİKETLER

Kalın metin = < b> < /b> (bold)
Eğik metin = < i> < /i> (italic)
Altı çizili = < u> < /u> (underline)
Üstü çizili = < s> < /s> (strike)
Satır atlama = < br/>
Sınırsız Çizgi = < hr/>
Yorum satırı = < !-- --> (comment)
Paragraf = < p> < /p> (pharagraph)
Bağlantı = < a> < /a> (link)
Resim = < img />

3 - RESİMLER

Resim Dosyasının Kaynağını Belirtme = src="url/dosya adı"
(dosya olarak belirtmek için, resim dosyasının html dosyasının bulunduğu klasörde olması gerekir)
Görüntülenemeyen Resim Açıklaması = alt="text"
Fare ile Resimin Üzerine Gelindiğinde Resim Açıklaması Vermek = title="text"

4 - LİNKLER

Linki Yeni Sekmede Açmak = target="_blank"

5 - TABLOLAR

< table> < /table> = tablo oluşturma
< thead> < /thead> = tablo başlık kısmı
< tbody> < /tbody> = tablo gelişme kısmı (bilgiler)
< tfoot> < /tfoot> = tablo sonuç kısmı (toplamlar)

border="1px" = tablo çizgileri ve kalınlığı
< th> < /th> = başlık eki (thead ve tfoot içerisinde)
< tr> < /tr> = satır oluşturma (tbody içerisinde)
< td> < /td> = sütun oluşturma (satır içerisinde)
colspan="2" = sütun birleştirme (yazıyı ortalama)
align="left-right" = satır içi hizalama
