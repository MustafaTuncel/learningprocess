5 - TABLOLAR
  • < table> < /table> = tablo oluşturma
  • < thead> < /thead> = tablo başlık kısmı
  • < tbody> < /tbody> = tablo gelişme kısmı (bilgiler)
  • < tfoot> < /tfoot> = tablo sonuç kısmı (toplamlar)
  • border="1px" = tablo çizgileri ve kalınlığı
  • < th> < /th> = başlık eki (thead ve tfoot içerisinde)
  • < tr> < /tr> = satır oluşturma (tbody içerisinde)
  • < td> < /td> = sütun oluşturma (satır içerisinde)
  • colspan="2" = sütun birleştirme
  • rowspan="2" = satır birleştirme
  • align="left-right" = satır içi hizalama

6 - LİSTELEME
  • < ol> < /ol> = Ordered liste oluşturma
  • < ul> < /ul> = Unordered liste oluşturma
  • < li> < /li> = Liste maddeleri
  • [ol/li/ul/li] = kategori - iç içe liste oluşturma (sıralı ve sırasız listeleme kullanarak)
  • < dl> < /dl> = Kategori - İç içe liste oluşturma
  • < dt> < /dt> = Kategori - Liste adı
  • < dd> < /dd> = Kategori - Liste özellikleri yada detayları
  • Unordered listelerde < ul> type="..."> type parametresi girilerek, (disc,circle,square gibi)
  • Ordered listelerde < ol> type="..."> type parametresi girilerek, (I,i,A,a gibi)
  • Ordered listelerde < ol> start="2,3,4..."> start parametresi girilerek (başlangıç maddesi) değişiklikler yapılabilir.

7 - FORM ETİKETLERİ
    a - Input [action, id, class, style, lang, name, value, type (text, password, email, checkbox, radio, submit...)]

      Öncesinde açıklama belirtilerek input isimlendirilebilir (ID < input type="text" /> gibi).
      Birden çok 'radio'ya aynı 'name' tanımlanırsa (sinema,tiyatro,spor...>hobi gibi) tek seçenek işaretlenebilir.
    1. < form> < /form> = verilerin gönderilip/alınabilmesi için kullanılır
    2. < required /> = zorunluluk uyarısı gösterir
    3. < action="..." /> = Submit'e tıklandığında formun gönderileceği konumu belirler (/Home/Register gibi).
    4. < input id="..." /> = eşsiz kimliğini belirtir
    5. < input class="..." /> = sınıf adını belirtir
    6. < input style /> = CSS kuralları tanımlar
    7. < input lang="..." /> = içerdiği metnin lisanını belirtir
    8. < input name="..." /> = kontrolün ismini belirtir
    9. < input value="..." /> = girilen verileri kaydetme ve otomatik tamamlama
    10. < input type="..." /> = girilen veri değerleri
    11. < input min="0" /> = en az girilmesi gereken veri sayısı
    12. < input max="50" /> = en fazla girilmesi gereken veri sayısı
    13. < input step="10" /> = arttırma/azaltma miktarı
    14. < input placeholder="..." /> = metin kutusu önizleme açıklaması
