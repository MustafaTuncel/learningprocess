CSS Descriptions

CSS KULLANIM ŞEKİLLERİ
    CSS dört şekilde kullanılabilir :

    1 - Sayfada Tanımlama

      Head bloğu içerisinde yazılan < style> tanımlamalarıdır.
      < head>
      < style> body{color: pink;} < style>
      < /head>       gibi.

    2 - InLine Tanımlama

      Satır içerisinde yazılan < style> tanımlamalarıdır.
      < p id="ilkparagraf" style="border: 1 solid #555" ;> Lorem, ipsum dolor sit amet consectetur adipisicing elit. Quas, quasi? < /p> gibi.

    3 - Ek Bir Dosyayı Sayfaya Tanımlama

      Projenin bulunduğu klasör içerisinde style.css adında bir dosya oluşturulur.
      CSS tanımlamaları bu dosyanın içerisine yazılır ve dosya içerisinde tekrardan < style> tagi kullanılmaz.

      Daha sonra çalışılan dosya içerisinde style.css dosyasıyla şu şekilde bağlantı kurulur ;
      < link rel="stylesheet" href="style.css">

    4 - Dosya Üzerinden Tanımlanarak


SAYFA İÇİ DÜZEN VE ETİKETLER

  • < style> etiketiyle oluşturulur.
  • Yapısı şu şekildedir ; seçici {özellik:değer; özellik2:değer} body {color:blueviolet} gibi.
  • Oluşturulmuş id'yi seçmek için #id adı yazılır.
  • Oluşturulmuş class'ı seçmek için .class adı yazılır.
  • Değerden sonra girilen !important tagi seçili attribute'e verilen değeri diğer genel tanımlamalardan üstün kılar.
    #ilkparagraf yada .ilkbaşlıkparagrafları {font-size: 40px!important;} gibi.
  • * tagi tüm sayfaya etki eder.


CSS STİLLERİ
    Background

    • background-image: url ("")
    • background-repeat: repeat, no-repeat, repeat X (yatay), repeat Y (dikey)
    • background-color: renk adı / kodu

    Text

    • Color: renk adı, kodu, rgb kodu
    • Text-align: left, right, center, justify, start, end, inherit, match-parent
    • Text-decoration: none
    • Text-transform: none, lowercase, uppercase, capitalize, full-width, full-size-kana, inherit
    • Text-indent: 50px, 10% (paragraf girişi)
    • Line-height: 25px (satırlar arası boşluk)
    • Letter-spacing: 1px
    • Word-spacing: 5px

    Link

    • a:visited {color: renk} (önceden ziyaret edildiğindeki renk)
    • a:link {color: renk} (normal link rengi)
    • a:hover {color: renk} (üzerine gelindiğindeki renk)
    • a:active {color: renk} (tıklandığındaki renk)

    List

    • .classadı {list-style-type: disc, circle, square} (liste simgesi değiştirme)
    • .classadı {list-style-image: url (image)} (liste simgesini özelleştirme)
    • .classadı {list-style-position: inside, outside} (liste girintisi)

    Table

    • table, th, tr, td {border: 2px solid #666} (tablo çizgilerini oluşturma)
    • table {border-collapse: collapse} (tablo çizgilerini birleştirme)
    • th, td {padding: 5px} (border-text uzaklığı)
    • tr td:nth-child (3) {text-align: center} (belirli sütunu hizalama)
    • tr td:nth-child (1) {background-color: yellow} (belirli sütuna arka plan rengi verme)
    • tr td:nth-child (2) {font-weight: bold} (belirli sütundaki içerikleri kalınlaştırma)
    • tbody tr:nth-child (odd) {background-color: aliceblue} (tek sayılı satırlara arka plan rengi verme)
    • tbody tr:nth-child (even) {background-color: greenyellow} (çift sayılı satırlara arka plan rengi verme)
    • tbody tr:first-child {font-style: italic; color: red} (ilk satır fontu-rengi değiştirme)
    • tbody tr:last-child {font-style: italic; color: red} (son satır fontu-rengi değiştirme)
    • *{font-family: "open sans", Arial, sans-serif} (tüm sayfanın fontunu değiştirme)
    • tbody tr:hover{background-color: yellow} (satır üzerine gelindiğinde renk değiştirme)

    Box
    div = box
    padding = iç boşluk
    margin = dış boşluk

    • < div class="classadı"> < /div>
      • width: 200px
      • background-color: burlywood
      • padding: 5px
        • padding-left: 10px
        • padding-right: 10px
        • padding-top: 10px
        • padding-bottom: 10px
        • padding: 10px 20px 30px 40px (saat yönünü ifade eder: 10px=12, 20px=3, 30px=6, 40px=9)
        • padding: 20px 40px (20px=yukarıdan aşağı, 40px=soldan sağa)
      • margin: 20px
        • margin-left: 10px
        • margin-right: 10px
        • margin-top: 10px
        • margin-bottom: 10px
        • margin: 10px 20px 30px 40px (saat yönünü ifade eder: 10px=12, 20px=3, 30px=6, 40px=9)
        • margin: 20px 40px (20px=yukarıdan aşağı, 40px=soldan sağa)
