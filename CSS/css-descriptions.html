CSS Descriptions

CSS KULLANIM ŞEKİLLERİ
    CSS üç şekilde kullanılabilir :

    1 - Internal CSS - Sayfa İçi Tanımlama

    Yazılan CSS kodları sadece tek bir sayfayı ilgilendiriyorsa kullanılır.

      Head bloğu içerisinde yazılan < style> tanımlamalarıdır.
      < head>
      < style> body{color: pink;} < style>
      < /head>       gibi.

    2 - InLine CSS - Satır İçi Tanımlama

    Yazılan CSS kodları sadece tek bir satırı ilgilendiriyorsa kullanılır.
    InLine yazılan CSS kodları Internal CSS kodlarını ezer.

      Satır içerisinde yazılan < style> tanımlamalarıdır.
      < p id="ilkparagraf" style="border: 1 solid #555" ;> Lorem, ipsum dolor sit amet consectetur adipisicing elit. Quas, quasi? < /p> gibi.

    3 - External CSS - Sayfa Dışı Tanımlama

    Birden çok html dosyasına CSS ile etki etmek istendiğinde kullanılır.
    Internal yada InLine yazılan CSS kodları External CSS kodlarını ezer.

      Projenin bulunduğu klasör içerisinde style.css adında bir dosya oluşturulur.
      CSS tanımlamaları bu dosyanın içerisine yazılır ve dosya içerisinde tekrardan < style> tagi kullanılmaz.

      Daha sonra çalışılan html dosyası içerisinde style.css dosyasıyla şu şekilde bağlantı kurulur ;
      < link rel="stylesheet" href="style.css">


CSS YAPISI

    CSS Tanımı

    • selector = seçici
    • declaration = ifade/ler
      • property = nitelik, özellik
      • value = değer

      selector declaration
      declaration = {property: value;}

      seçici {özellik:değer; özellik2:değer}
      body {color: blueviolet;}

    CSS Selectors
      Group Selectors:
      • * = tüm etiketler
      • p = tüm paragraf etiketleri
      • div.classadı = belli sınıftaki divleri seçer(divin kendisi)
      • div .classadı = belli sınıfları seçer(divin altındaki sınıflar)
      • div p = div içindeki tüm p etiketleri
      • div>p = sadece üst etiketi div olan p etiketleri
      • div,p = tüm div etiketleri ve tüm p etiketleri
      • p~div = p ile aynı seviyedeki div etiketleri ve p etiketlerinden sonra gelen div etiketleri
      • p+div = p etiketinden sonra gelen ilk div etiketleri

      Attribute Selectors:
      • p[title] {border: 1px solid gray; padding: 5px;} p elementlerindeki title attributelerinin tümünü seçer
      • p[title="third paragraph"] {background-color: gold;} p elementlerindeki title attributelerinde geçen (third paragraph) title adını seçer
      • p[title~"third"] {background-color: red;} p elementlerindeki title attributelerinin içinde geçen (third) değerin bulunduğu titleları seçer (= yerine ~ kullanılır)
      • a[href^="http"] {color: red;} a elementlerindeki href attributelerinin başında geçen (http) değerin bulunduğu linkleri seçer
      • a[href$="com"] {color: aqua;} a elementlerindeki href attributelerinin sonunda geçen (com) değerin bulunduğu linkleri seçer
      • a[href="https://www.example.net"][target="_blank"] {color: yellow;} a elementlerindeki href ve target attributeleri içinde geçen (https://www.example.net ve _blank) değerlerin bulunduğu linkleri seçer
      • input[type="text"] {display: block; width: 150px; background-color: red;} input elementlerindeki type attributelerinin içinde geçen (text) değerin bulunduğu inputları seçer
      • input[type="button"] {background-color: green; color: white; padding: 8px 16px;} input elementlerindeki type attributelerinin içinde geçen (button) değerin bulunduğu inputları seçer
      • img[src="https://www.example.com"] {color: orange;} img elementlerindeki src attributelerinde geçen (https://www.example.com) değerin bulunduğu resimleri seçer
      • div[id="example1993"] div elementlerindeki id attributelerinde geçen (example1993) değerin bulunduğu div bloklarını seçer

      Pseudo Selectors
      Sayfa içerisindeki elementlerin bir parçasını seçmek istediğimizde Pseudo Elementlerini kullanırız.
      Sayfa içerisindeki elementlerin altında bulunan attributeleri seçmek istediğimizde Pseudo Classlarını kullanırız.

        Pseudo Elements
        • p::first-line {font-weight: bold; color: brown; word-spacing: 2px; letter-spacing: 1px;}
        • p::first-letter {font-size: 48px;}
        • p::before {content: "-deneme"; color: blueviolet; background-color: yellow; width: 60px;}
        • p::after {content: "-deneme"; color: blueviolet; background-color: yellow; width: 60px;}

        Pseudo Classes
        • ul li {padding: 10px; margin: bottom 10px; list-style-type: none; width: 450px; border-radius: 50px;}
        • ul li:nth-of-type(odd) {border: 2px solid blue; margin: 5px; background-color: green; color: white;}
        • ul li:nth-of-type(even) {border: 2px solid red; margin: 5px; background-color: purple; color: white;}
        • ul li:first-child {background-color: yellow; color: blue;}
        • ul li:last-child {background-color: yellow; color: red;}
        • a:link {color: goldenrod;}
        • a:hover {color: green;}
        • a:active {color: red;}
        • a:visited {color: purple;}
        • div:hover span {display: block;}

      CSS Inherit

      a elementini baz alırsak ; kendisi üzerinde bulunan elementlerden (body>ul>li>a vb.) css kodu almaz.
      Bu türdeki elementlerin üst elementlerden attribute değerlerini alabilmeleri için inherit kodu kullanılır.

      < div id=example yada class=example >< /div>
      #example span {color: inherit;} yada
      .example span {color: inherit;} gibi


SAYFA İÇİ DÜZEN VE ETİKETLER

  • < style> etiketiyle oluşturulur.
  • Oluşturulmuş id'yi seçmek için #id adı yazılır.
  • Oluşturulmuş class'ı seçmek için .class adı yazılır.
  • Değerden sonra girilen !important tagi seçili attribute'e verilen değeri diğer genel tanımlamalardan üstün kılar.
    #ilkparagraf yada .ilkbaşlıkparagrafları {font-size: 40px!important;} gibi.
  • cursor: pointer (tanımlandığında imleç türünü değiştirir)
  • focus: {background-color: lightgoldenrodyellow;} (tanımlandığında, öğeye odaklanıldığında gerçekleşecek eylemi belirtir)

    • tbody tr:last-child {font-style: italic; color: red} (son satır fontu-rengi değiştirme)

    • tbody tr:hover{background-color: yellow} (satır üzerine gelindiğinde renk değiştirme)

      • position: sticky (scroll bar hareket ettirildiğinde, divin belirtildiği pixeldeki noktaya gelindiğinde div belirtilen konumun başlangıcına yapışır / top: 0px gibi)
