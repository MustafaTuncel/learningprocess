Javascript'e Giriş

  • alert(""); = web sayfası mesajı
  • console.log(); = web tarayıcısı konsolunda log göstermek
  • console.warn(""); = web tarayıcısı konsolunda uyarı mesajı göstermek
  • console.error(""); = web tarayıcısı konsolunda hata mesajı göstermek
  • console.clear(); = web tarayıcısı konsolunda geçmişi temizler
