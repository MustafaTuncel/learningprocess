JAVASCRIPT'E GİRİŞ
  1. Javascript dosyamızı html dosyası içerisinde yazmak yerine, ek bir script.js dosyası oluşturarak da yazabiliriz.
  2. Oluşturduğumuz script.js dosyamızı, html>body içerisine en altta olacak şekilde ;
    <.script src="script.js"> < /script> yazarak tanımlayabiliriz.
    Yada ;
    HTML ve CSS kodlarının ardından JS kodlarını çalıştırmak için html>head içerisine ;
    <.script src="script.js" defer> < /script> kodunu yazarak da dosyamızı tanıtabiliriz.
  3. Web tarayıcıları üzerinde, öğe denetleme kısmını açarak 'console' üzerinden JS kodlarımızı test edebiliriz.


DEĞİŞKENLER (variables)

    Önemli Noktalar ve Uyarılar

  1. Kodlar yukarıdan aşağıya doğru işletilir, sıra önemlidir !
  2. Önceden değer atanmış bir değişkene tekrar değer atanırsa ; ilk atanan değer silinir ve son atanan değer etkin olur.
  3. Değişken isimleri büyük - küçük harflere duyarlıdır ! (Case sensitive)
  4. Atama yapılmamış değişkenler konsolda 'undefined' olarak gösterilir.
  5. Değişken isimleri ;
    alpha numeric karakterler ile başlayabilir.
    sayı ile başlayamaz !
    komut adı ile başlayamaz !
  6. Bir değişken üzerinde birden fazla kelimeyi değişken ismi olarak kullanamayız ! (ad soyad vb. şekilde)
    Bunun yerine kelimelerin aralarında herhangi bir işaret kullanarak yada kelimeleri birleştirerek birden fazla kelimeyi değişken ismi olarak kullanabiliriz. (ad_soyad yada AdSoyad vb. şekilde)
  7. Atanmış değerlerde ;
    sayısal ifadeler mavi renkte, (number)
    sözel ifadeler gri renkte gösterilir (string)

    Kullanılan Terimler

  • var = tekrar tanımlanabilir ve değeri güncellenebilir değişken oluşturma komutu
  • let = tekrar tanımlanamaz ancak değeri güncellenebilir değişken oluşturma komutu (var ile oluşturulan tüm komutları oluşturabilir.)
  • const = tekrar tanımlanamaz ve değeri güncellenemez değişken oluşturma komutu
  • age = değişken ismi
    fullname = değişken ismi
    isactive = değişken ismi
  • 20 = değişkene atanan değer (sayısal değerler tırnak içine alınmaz)
    "Mustafa Tuncel" = değişkene atanan değer {(stringler) sözel değerler tırnak içine alınır}
    True - False = değişkene atanan değer (boolean değerleri tırnak içine alınmaz)
  • console.log(değişkenismi); = web tarayıcısı konsolunda log göstermek
  • console.log(typeof 'değişkenismi'); = 'typeof' parametresi, web tarayıcısı konsolunda değişkenin tipini gösterir (string, number, array, boolean...)
  • console.warn("uyarımesajı"); = web tarayıcısı konsolunda uyarı mesajı göstermek
  • console.error("hatamesajı"); = web tarayıcısı konsolunda hata mesajı göstermek
  • console.clear(); = web tarayıcısı konsolunda geçmişi temizler
  • alert("websayfasımesajı"); = web tarayıcısında web sayfası açıldığında mesaj göstermek


DEĞİŞKEN TİPLERİ (variable types)

  • Pritive Types (tekil kullanıma izin verir)

    • Undefined = let car; (değer atanmamış)
    • String = let firstname = "Mustafa"; (sözel)
    • Number = let age = 29; (sayısal)
    • Boolean = let IsActive = true; (doğru - yanlış, aktif - pasif, açık - kapalı)
    • Null = let job=null; (değer atanmayacak değişkene atanan değerdir, hiç bir değeri temsil etmeyen boşluğu temsil eden bir değerdir)

  • Reference Types - Objects (çoğul kullanıma izin verir)

    • Array = let names = ["Ali", "Ahmet", "Can"];(tek bir değişken altında, birden fazla verinin saklanması için kullanılır)
    • Object Literals = let address = {city: "Tekirdağ", country: "Türkiye"}; (obje / nesne oluşturmak için kullanılır)
    • Function = (işlem yapmayı sağlayan kod bloğu oluşturmak için kullanılır)


TÜR DÖNÜŞTÜRME (type conversions)

  • String string
  • Number number
  • Boolean boolean

  • Number(true); = 1
  • Number(false); = 0
  • Number(null); = 0
  • Number("a"); = NaN (not a number)
  • Number([1,2,3,4]); = NaN (not a number)

  • parseInt("10"); = Ondalıklı sayıları tam sayıya yuvarlar
  • parseFloat("10.50"); = Ondalıklı sayıları olduğu gibi görüntüler

  • string to number = Sözel değeri sayısal değere dönüştürmek

    let num1 = ("5");
    let num2 = ("10");
    console.log(num1+num2);     dediğimizde, değişkenler string değerli oldukları için toplandıklarında yanyana gelirler ve sonuç "510" olur.

    let num1 = Number("5");
    let num2 = Number("10");
    console.log(num1+num2);     yada
    let total = num1+num2;
    console.log(total);     dediğimizde, değişkenler sayısal değere dönüştürüldükleri için toplandıklarında yanyana gelmezler, matematik işlemi gerçekleşir ve sonuç "15" olur.

  • number to string = Sayısal değeri sözel değere dönüştürmek

    val = 10
    val = String(10);
    console.log(val);     dediğimizde, değişken sözel değere dönüştürülmüş olur.

  • boolean to string = Boolean değerini sözel değere dönüştürmek

    val = true
    val = String(true);     dediğimizde, değişken sözel değere dönüştürülmüş olur.

  • date to string = Date değerini sözel değere dönüştürmek

    val = new Date();
    val = String(new Date());     dediğimizde, değişken sözel değere dönüştürülmüş olur.

  • array to string = Array değerini sözel değere dönüştürmek

    val = String([1,2,3,4]);     dediğimizde, değişken sözel değere dönüştürülmüş olur.

  • toString() = Arkasına eklendiği değeri sözele dönüştürür

    val = (10).toString();     yada
    val = (false).toString(); dediğimizde, değişken sözel değere dönüştürülmüş olur.


OPERATÖRLER (operators)

    let val;
    const a = 10;
    const b = 5;
    const c = 5;
    let d = 3;     dediğimizi varsayarsak
  1. Aritmetik Operatörler (Matematiksel İşlemler)

    2. val = a + b;
    val = a - b;
    val = a * b;
    val = a / b;
    val = a % b; //mod almak, sayının tek mi çift mi olduğunu anlamak için sayıyı ikiye böler (0 = çift, 1 = tek)
    val = d++; // değişkeni konsola yazdırdıktan sonra arttırma işlemi yapar, önceki değer gözükür
    val = ++d; // değişkeni konsola yazdırmadan önce arttırma işlemi yapar, güncel değer gözükür
    val = d--;
    val = --d;

  2. Atama Operatörleri (Matematiksel Kısayol İşlemler)

    3. val = a;
    val += a; // val = val + a (10+10)
    val -= a; // val = val - a (20-10)
    val *= a; // val = val * a (10*10)
    val /= a; // val = val / a (100/10)
    val %= a; // val = val % a (10/2 = 0, val çift sayı)

  3. Karşılaştırma Operatörleri (Eşitlik, Büyüklük - Küçüklük ve Tip Kontrolleri)

    4. val = a == b; // false
    val = b == c; // true
    val = a == b + c; // true

    val = 5 == "5"; // true, sadece değeri kontrol eder (birisi Number birisi String olmasına rağmen doğru sonuç verir)
    val = 5 === "5"; // false, değeri ve tipi kontrol eder (birisi Number Birisi String olduğu için yanlış sonuç verir)

    val = a!=b; // true, a b'ye eşit değildir
    val = a!==b; // true, a b'ye eşit değildir
    val = a > b; // true, a b'den büyüktür
    val = b < a; // true, b a'dan küçüktür val=a>= b; // true, a b'ye eşit yada daha büyüktür
    val = a <= b; // false, a b'ye eşit yada daha küçük değildir
    val = 5 >= 5; // true, 5 5'e eşit yada daha büyüktür
    val = 4 >= 5; // false, 4 5'e eşit yada daha büyük değildir

  4. Mantıksal Operatörler

    5. && (and) (birden fazla değerin sonucunun ayrı ayrı değil tek olarak alınması)

    // true && true = true
    // true && false = false
    // false && false = false

    val = a > b // true
    val = a > c // true
    val = (a > b) && (a > c) // true

    || (or) (birden fazla değerden herhangi birisinin sonucunun alınması)

    // true && true = true
    // true && false = true
    // false && false = false

    val = a > b // true
    val = a > c // true
    val = (a > b) || (a > c) // true

    ! (not) (ters değer alınması)

    // !true = false
    // !false = true

    val = a > b // true
    val = a > c // true
    val = !(a > b) // false
